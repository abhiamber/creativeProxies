
# Creative Proxy

## Tech Stack

ReactJS, JavaScript, Chakra-UI,


## Deployment Link
https://createive-proxies.vercel.app/

## Installation

Clone the Repository from Github. Then do the following steps:

```bash
    npm install

    npm run start
```
